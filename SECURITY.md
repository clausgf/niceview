Security Policy
===============

Supported versions
------------------

NiceView is pre-1.0 software; only the latest released version receives fixes.

Reporting a vulnerability
-------------------------

Please report security issues privately rather than via a public issue. Use GitHub's
[private vulnerability reporting](https://github.com/clausgf/niceview/security/advisories/new)
for this repository, or email the maintainer at <fuehner@gmx.net>.

Please include a description of the issue, the affected version, and — if possible — a minimal
reproducer. You can expect an initial response within a reasonable time frame; there is no bug
bounty program.
